# Educational RAG Assistant - Project Documentation

## 1. Introduction
The Educational RAG Assistant is an AI-powered system designed to help students learn more effectively by answering questions based on uploaded study materials using RAG.

## 2. Problem Statement
Students face difficulty in searching information from large notes and books. This system solves that problem using AI.

## 3. Objectives
- Provide accurate answers
- Save time
- Improve learning
- Easy to use system

## 4. System Architecture
1. Upload Documents
2. Generate Embeddings
3. Store in Vector Database
4. User Query
5. Retrieve Data
6. Generate Answer

## 5. Technologies Used
- Python
- LangChain
- FAISS / ChromaDB
- HuggingFace / OpenAI
- Streamlit / Flask

## 6. Workflow
1. Upload files
2. Process data
3. Store embeddings
4. Ask question
5. Get answer

## 7. Installation
Clone repo:
git clone https://github.com/shru-354/educational-rag-assistant.git

Install packages:
pip install -r requirements.txt

Run:
python app.py

## 8. Advantages
- Fast response
- Easy learning
- Saves time

## 9. Limitations
- Internet required
- Depends on document quality

## 10. Future Scope
- Mobile app
- Voice support
- Multi-language
- Cloud hosting

## 11. Conclusion
This project helps students learn efficiently using AI-based document retrieval.
